# image-to-ppt-skills

A Codex skill for converting reference images, paper figures, and generated visual blocks into editable, validated PowerPoint slides.

## Goal

Use `$image-to-ppt-skills` to recreate a reference effect image, paper figure, or generated image-element set as a PowerPoint slide/deck with high visual fidelity.

## Required reconstruction behavior

- Keep visual blocks complete (do not omit panels, insets, callouts, or grouped regions).
- Preserve charts with their legends, metrics, captions, and labels.
- Rebuild layouts with precise positioning and spacing.
- Keep all text editable in PowerPoint.
- Recreate boxes, arrows, connectors, and other diagram primitives as editable shapes.
- Provide validation previews to compare recreated slides against source visuals before final delivery.
